# MiniRT

## This is a RayTracer in C
Made for UNIX systems, Linux and MacOs
- With implementation of fixed point numbers
- Reflexions
- Many shapes
    - Spheres
    - Cylinders
    - Planes
    - ...
---
## Compile
`$>make` in main repo
---
### With Graphic libraries GLFW and MLX42 from [Codam](https://github.com/codam-coding-college/MLX42)
