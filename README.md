# MiniRT

This is a RayTracer in C
- With implementation of fixed point numbers
- Reflexions
- Many shapes
    - Spheres
    - Cylinders
    - Planes
    - ...

- Used with Graphic libraries GLFW and MLX42 from [Codam](https://github.com/codam-coding-college/MLX42)
